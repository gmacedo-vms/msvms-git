##Git push

> git push origin nombre-rama

Envia los cambios locales al repositorio ubicado en github en la rama indicada (nombre-rama).

> git push -u origin nombre-rama

Envia los cambios locales al repositorio ubicado en github en la rama indicada (nombre-rama).
