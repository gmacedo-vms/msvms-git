
----------------------------------------------------------
Git
----------------------------------------------------------
- Software control de versiones.
- Creado por Linus Torvalds
- Similar a Subversion (TortoiseSVN)

----------------------------------------------------------
Ventajas
----------------------------------------------------------
- Auditoria de codigo
- Control de cambios en el tiempo
- Uso de etiquetas para versionar
- Trabajo en equipo

----------------------------------------------------------
Desventajas
----------------------------------------------------------
- Curva de aprendizaje
- Control de archivo diferentes a txt

----------------------------------------------------------
Git y GitHub
----------------------------------------------------------
- Git, Sistema de control de versiones
- GitHub (applicación), red social que utiliza Git para administrar nuestros proyectos
-Entonces Git también es diferente a Bitbucket y a GitLab
